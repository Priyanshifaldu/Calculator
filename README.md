# Java Calculator

A simple console-based Calculator application developed in Java that performs basic arithmetic operations. This project demonstrates fundamental Java programming concepts, object-oriented design, and modular application development.

---

## Overview

This project provides a lightweight calculator capable of performing basic mathematical operations through a command-line interface.

The application was developed to strengthen understanding of:

* Java Programming
* Object-Oriented Programming (OOP)
* User Input Handling
* Mathematical Computations
* Modular Design

---

## Features

* Addition of numbers
* Interactive console-based execution
* Modular code structure
* Easy to understand and extend
* Beginner-friendly implementation

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

## How to Run

### Compile

```bash
javac *.java
```

### Execute

```bash
java Calculator
```
